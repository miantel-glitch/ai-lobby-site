## WHERE THIS GOES IN THE CODE

Once you've filled this out, the information gets added to:

1. **`data/characters.json`** - Basic character config (traits, relationships, triggers, etc.)
2. **`netlify/functions/ai-perplexity.js`** (or a new file) - Full system prompt for voice generation
3. **`netlify/functions/breakroom-chatter.js`** - Breakroom personality
4. **`netlify/functions/chat.js`** - Add to employeeFlair and headshots objects
5. **`images/`** - Add headshot image
